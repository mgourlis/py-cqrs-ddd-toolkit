## [Unreleased]

### Added
- Initial project structure for `py-cqrs-ddd-toolkit`.
- Framework-agnostic CQRS and DDD building blocks.
- Declarative decorators for commands, queries, and event handlers.
- Multi-backend support (SQLAlchemy, Memory, Redis).
- Storage abstraction layer (Local, S3, Dropbox, GCS).
- Comprehensive test suite for all backends.
- Pre-commit hooks for ruff and mypy.
- GitHub Actions CI workflow.
